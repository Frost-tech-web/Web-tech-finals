# Brew & Bean Coffee Shop Website
**Web Technology Finals Project**

## Project Description
A responsive website for a fictional local coffee shop called "Brew & Bean". Built as a final project for Web Technology class. The site includes a homepage, about section, interactive menu, photo gallery slider, and a contact form with validation.

---

## Technologies Used
- **HTML5** - Semantic tags (header, main, section, footer)
- **CSS3** - Flexbox, CSS Grid, Media Queries, CSS Variables
- **JavaScript (Vanilla)** - No frameworks; just plain JS for interactivity
- **Google Fonts** - Playfair Display + Lato

---

## Features
- Responsive navigation with hamburger menu for mobile
- Hero section with call-to-action button
- Tabbed menu section (Coffee / Pastries / Cold Drinks)
- Auto-playing image slider with prev/next controls and dot indicators
- Contact form with client-side validation
- Mobile-first responsive design
- WCAG-friendly color contrast and alt text on all images

---

## How to Run Locally
1. Download or clone this repository
2. Open `index.html` in any web browser (no server needed!)

If you have VS Code with Live Server:
Right-click index.html > "Open with Live Server"
